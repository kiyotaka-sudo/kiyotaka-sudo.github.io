# Portfolio Étudiant

## Présentation
Site web personnel présentant mon parcours, mes compétences techniques et mes projets réalisés dans le cadre de ma formation à KEYCE Academy.

## Technologies utilisées
- HTML5
- CSS3 (avec variables CSS et design responsive)
- JavaScript (manipulation du DOM, localStorage)

## Fonctionnalités

- Comptage et persistance des vues via localStorage
- Design moderne et sombre avec animations
- Structure responsive adaptée à tous les appareils
