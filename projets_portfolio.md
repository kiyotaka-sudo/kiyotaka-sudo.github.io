# Projets Académiques & Réalisations (Portfolio)

Ce document regroupe l'ensemble des projets réalisés durant mon parcours d'études en informatique, classés par domaine de compétence avec leurs descriptifs et stacks techniques associés.

## 📱 1. Développement Mobile & Web Full-Stack

### **CityShare (ex-KmerPool)**
*   **Description :** Application mobile de covoiturage et de livraison de colis collaboratif pour le Cameroun.
*   **Stack Technique :** React Native (Expo SDK 54), Spring Boot, PostgreSQL, JWT, Firebase Cloud Messaging, WebSockets/STOMP, OpenStreetMap, Monetbil (MTN/Orange Mobile Money).

### **KeyceBet**
*   **Description :** Application de paris sportifs pour le marché camerounais (inspirée de BetPawa).
*   **Stack Technique :** Spring Boot, React, PostgreSQL/Redis, Docker, Flyway, JWT, WebSockets.

### **Plateforme Green IT & IA environnementale**
*   **Description :** Projet de sensibilisation et de calcul de l'empreinte écologique.
*   **Stack Technique :** React, Recharts, API Claude.

### **Portfolio Web interactif**
*   **Description :** Site web vitrine moderne de présentation personnelle et de réalisations.
*   **Stack Technique :** HTML5, CSS3, JavaScript, LocalStorage.

### **Application Web de gestion de livres**
*   **Description :** Plateforme de gestion complète d'une bibliothèque avec interface dynamique.
*   **Stack Technique :** PHP, MySQL, AJAX, XML, HTML5, CSS3, JavaScript.

### **Déploiement d’un site Web**
*   **Description :** Mise en ligne et configuration opérationnelle d'applications web.
*   **Stack Technique :** PHP, MySQL, InfinityFree, DNS, Hébergement Web.
